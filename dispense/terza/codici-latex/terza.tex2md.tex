
    \begin{itemize}
        \item \textbf{Valore}: dato un numero.
        \item \textbf{Rappresentazione}: un dato valore può essere rappresentato in più modi (ad esempio base 2, base 8, base 10).
        \item \textbf{Codice}: un dato valore viene rappresentato in una forma compatibile ad uno specifico scopo.
    \end{itemize}
